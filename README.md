# EduOS

EduOS is an education-focused operating system for website publishing, YouTube production, and evidence-grounded review workflows.

## Current Foundation

This repository currently starts from the StudioOS scaffold, with selected ResearchAgent-style tooling already ported:

- OpenRouter-first model routing with persisted agent and tier overrides
- Model catalog refresh and routing settings endpoints
- Writing, media, promo, memory, and workflow orchestration surfaces
- Project-based artifact persistence and pipeline execution

## Accuracy Direction

EduOS is being hardened for educational accuracy using GrantOS-style rigor:

- research briefs are expected to separate established facts from hypotheses and open questions
- script generation is instructed to preserve uncertainty instead of flattening it into false certainty
- media scripts now pass through an `accuracy_reviewer` gate before SEO, thumbnail, audio, and publish packaging
- a reusable strict educational accuracy policy is available in the prompt library
