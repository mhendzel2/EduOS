# Evidence-Based Pedagogy Framework

## Theoretical Principles
1. **Mayer's Multimedia Guidelines**:
   - **Coherence Principle**: Strip all unrelated fun facts and visual clutter. Do not distract with purely aesthetic decorations in science breakdowns.
   - **Modality Principle**: People learn better from graphics & narration than from graphics, narration, & redundant on-screen text. Do not generate scripts that rely on massive on-screen text blocking.
2. **Cognitive Load (Sweller)**:
   - Chunk intense biomolecular sequences. If explaining phase separation, start with polymer physics visually BEFORE introducing multi-valency mathematics.
3. **Bloom's Alignment**:
   - Ensure script questions force the viewer upward: Knowledge $\rightarrow$ Comprehension $\rightarrow$ Application $\rightarrow$ Synthesis.
