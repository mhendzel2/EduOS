# Animated Diagram Specifications

## Temporal Animation Syntax
1. **Directionality & Scaling**:
   - Define exact bounds: "PAN LEFT from the nuclear envelope surface towards internal lamina heterochromatin structure."
   - Frame transition: "T = 0.5s: Fluorescent focus bleaches to 0. T = 2.0s: Focus recovers 40% intensity smoothly."
2. **Morphology vs Velocity**:
   - Never rely on static spatial coordinates. Emphasize molecular vibration (Brownian dynamics vs targeted walk). Example: "Molecule X randomly jitters (amplitude 2) until docking receptor Y, slowing to rigid lock."
