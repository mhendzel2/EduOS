# Science Misconception Database

## Common Fallacies to Flag
1. **Condensation**:
   - *Misconception*: Droplets behave like oil-in-water physically isolating DNA into rigid droplets.
   - *Correction*: Liquid-liquid condensates act as dynamic selective permeability barriers heavily interconnected via long polymer tails.
2. **Chromatin Structure**:
   - *Misconception*: Heterochromatin is perfectly condensed "dead" territory.
   - *Correction*: Heterochromatin actively loops and maintains necessary architectural rigidity that anchors the entire nucleus; transcription factors structurally sample it continually.
3. **DNA Replication Directionality**:
   - *Misconception*: Polymerases smoothly copy the entire genome like a zipper.
   - *Correction*: Must emphasize Okazaki fragmentation limits, torsional supercoiling, and topological collision limits.
